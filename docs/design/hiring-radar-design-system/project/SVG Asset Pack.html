Hiring Radar · Brand

SVG Asset Pack

The only illustrations allowed in Hiring Radar screens. Every asset is a self-contained, stroke-based SVG — technical and precise, blueprint-meets-radar-scope. 1.5px stroke throughout (favicon excepted), no gradients beyond phosphor glows, no people, no emoji. Reference each by its name.

Hardcoded system hex (no CSS-var dependency), 1.5px stroke, embedded glow filters — drop any of these into a screen with <img src="assets/illustrations/<name>.svg"> or inline. The loading-sweep and blip-sprite are meant to be animated (CSS rotate / ping); honor prefers-reduced-motion at the usage site.
